##  Project Overview
#### This project aims to develop an intelligent stock trading bot using deep reinforcement learning within the FinRL environment. 
#### The bot will actively trade stocks, manage portfolios, and implement highfrequency trading methods using cuttingedge technology and deep reinforcement learning strategies.

##  Motivation
#### The objective behind this project is to investigate the potential of modern technologies, notably deep reinforcement learning, in the context of financial markets. 
#### The project aims to use artificial intelligence to construct a smart trading agent capable of generating abovemarket profits and effective risk management strategies.

## Goals

#### 1. Algorithmic Stock Trading: Create a bot that can make stock trading decisions autonomously based on learned patterns and market conditions.
#### 2. Portfolio Management: Optimize asset allocation and risk management strategies to help the bot manage portfolios more effectively.
#### 3. HighFrequency Trading Strategies: Implement and improve highfrequency trading tactics to capitalize on shortterm market inefficiencies and acquire a competitive advantage.
#### 4. AboveMarket Gains: Aim for persistent abovemarket gains by constantly learning and changing trading tactics utilizing deep reinforcement learning techniques.
#### 5. Effective Risk Management: Implement strong risk management measures to reduce possible losses and preserve the stability of trading operations.

## Key Components
#### FinRL Environment: Framework for reinforcement learning in finance.
#### Deep Reinforcement Learning Strategies: Techniques for adaptive decisionmaking.
#### Stock Trading Module: Algorithms for realtime trading decisions.
#### Portfolio Management System: Framework for managing portfolios with risk and return objectives.
#### HighFrequency Trading Tactics: Strategies for shortterm market opportunities.

 ### HOW TO RUN THE CODE

 #### Clone the Repository 
 #### install required libraries using 'pip install -r requirements.txt'
 #### start scraping data
 ##### for a single data - ###### testing purpose only use the data_preprocessing.py file runt the command 'python data_preprocessing.py' in the terminal
 ##### for all stocks in s&p 500 - run the command 'python data__preprocessing.py' in the terminal
 #### now move to train the model - LSTM use the command 'python lstm_model.py' in the terminal.
 #### for need to fetch the prediction value for the ddpg agent use the command ' python lstm_data_prep.py' in the terminal.
 #### now train the ddpg agent using the command 'python train_ddpg.py' in the terminal.
 #### run the main file to up the trading bot using the command 'python main.py' in the terminal.
